Joins the code to display variables

| Specifier | Meaning                           | Example        |
| --------- | --------------------------------- | -------------- |
| `%d`      | Integer (`int`)                   | `25`           |
| `%f`      | Floating-point (`float`/`double`) | `3.14`         |
| `%c`      | Single character                  | `A`            |
| `%s`      | String                            | `"Hello"`      |
| `%p`      | Pointer (memory address)          | `0x7ffee3b...` |
